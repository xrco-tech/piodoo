üöÄInstallation

  1. Copy the home-theme folder to your Odoo addons directory
  2. Restart your Odoo server
  3. Go to Apps menu and click Update Apps List
  4. Remove the Apps filter and search for "Enterprise Home Screen"
  5. Click Install
  6. Refresh your browser
  7. Navigate to the new Home menu item to see your dashboard!
